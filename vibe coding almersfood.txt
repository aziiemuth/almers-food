Buat website modern, clean, dan user friendly untuk frozen food dengan brand “Almers Food”. Website harus terasa premium tapi tetap hangat dan rumahan. Gunakan Next.js App Router (folder app) dengan React .jsx (bukan .tsx). Styling wajib memakai Styled Components (bukan Tailwind). Gunakan Swiper untuk hero slider, Framer Motion untuk animasi halus, Sonner untuk notifikasi, Lucide React untuk icon utama, React Icons untuk icon tambahan (misal sosial media, rating, kategori, tombol UI), Zustand untuk state cart/order, dan React Loading Skeleton untuk loading state. Tema warna wajib mengikuti referensi logo dengan nuansa elegan, lembut, dan premium untuk brand makanan. Gunakan gradient halus dan jangan terlalu ngejreng.

LOGIC CART WAJIB (PENJUMLAHAN OTOMATIS)
Saat user klik “Tambah Pesanan”:
•	Jika produk belum ada di cart: tambahkan sebagai item baru dengan qty = 1
•	Jika produk sudah ada di cart: qty bertambah (qty + 1)
Cart wajib menyimpan: productId, name, price, selectedVariant (jika ada), qty
User bisa:
•	Menambah qty dengan tombol plus
•	Mengurangi qty dengan tombol minus (jika qty jadi 0, hapus item)
•	Hapus item langsung
•	Clear cart

RINGKASAN PESANAN (KANAN DESKTOP / BAWAH MOBILE)
Wajib menampilkan:
•	List item (nama + varian) + qty
•	Harga satuan (price)
•	Subtotal per item = price x qty
•	Total keseluruhan = jumlah semua subtotal item (jika beli lebih dari 1 produk, total harus otomatis ditambah)
•	Tombol “Checkout via WhatsApp”
